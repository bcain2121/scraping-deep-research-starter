# Deep research workflow

Use this when a question needs more than a quick search.

## Phase 1: Normalize

Write:

- core question
- audience
- geography/timeframe
- deliverable format
- what is out of scope

## Phase 2: Plan

Create research tasks:

- official/primary sources
- expert or industry sources
- news/current sources
- opposing viewpoints
- data/statistics
- definitions and background

## Phase 3: Execute

Gather sources with:

- Tavily/web search for discovery
- direct scraping for specific pages
- small bounded crawls for site sections
- structured extraction from source text

## Phase 4: Verify

For each major finding:

- cite the source URL
- prefer primary sources
- compare against independent sources
- flag contradictions
- mark confidence level

## Phase 5: Synthesize

Deliver:

- executive summary
- key findings
- source-backed detail
- open questions
- bibliography

## Phase 6: Bounded improvement

Do only 1–2 follow-up passes focused on the biggest gaps. Avoid endless research loops.
