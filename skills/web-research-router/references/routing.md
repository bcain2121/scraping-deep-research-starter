# Routing details

## Search first when

- the user does not provide a URL
- you need current sources
- you need candidate official pages

## Scrape when

- the user provides a URL
- Tavily/search results identify a useful source
- you need page text, links, or quotes

## Crawl when

- one domain has many relevant pages
- the site has docs, blog categories, or directories
- you can bound the crawl safely

## Deep research when

- the task needs synthesis
- facts must be verified across multiple sources
- the deliverable should include findings and bibliography
